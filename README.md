# AI Engineering Codebasics Roadmap

A personalized 3-month intensive learning path designed to transition from software engineering to AI engineering roles at top companies like Brain Station 23, SELISE, or remote international positions.

## 🎯 Goal

Land an AI engineering / full-stack-with-AI role by the end of the 12-week sprint, building on existing SWE experience (~2 years), MERN/Next.js, Python, LangChain, LlamaIndex, vector databases, TensorFlow/PyTorch knowledge, and active portfolio projects.

## 📋 What This Roadmap Is

This is **NOT** a beginner-to-end generic AI curriculum. Instead, it's a compressed, re-sequenced version of a comprehensive source roadmap that:

- **Skips** content you already know (RAG basics, vector DBs, Python fundamentals)
- **Focuses** on gaps in your current skillset (eval/observability, formal agent patterns, MCP)
- **Integrates** directly with your live portfolio projects (CartQ SaaS, Bengali ESP32 TinyML project)
- **Prioritizes** skills actually demanded by target job markets (Azure, LangGraph, multi-agent systems)

## 🗓️ 12-Week Overview

### Weeks 1–2: Gap-Fill + Eval/Observability Foundations
- Docling for document parsing
- RAGAS for RAG evaluation
- LangSmith tracing and cost tracking
- **Output:** Instrumented CartQ RAG pipeline with measurable eval scores

### Weeks 3–5: Agentic Systems
- Week 3: Multi-agent systems with LangGraph
- Week 4: Model Context Protocol (MCP)
- Week 5: Bengali agent + eval harness (differentiator project)
- **Output:** Production-shaped multi-agent commerce system in CartQ

### Weeks 6–7: Deployment + Cloud
- Azure AI Foundry, Azure ML, Azure Functions (priority cloud)
- AWS AgentCore (secondary)
- Cost optimization: semantic caching, rate limiting, model cascading
- Multimodal RAG + open source contribution

### Week 8: Fine-Tuning + Local Models
- Time-boxed coverage of LoRA/QLoRA
- Ollama for local model deployment
- Optional: Small fine-tune for Bengali intent classification

### Weeks 9–10: Portfolio Polish
- Case study writing for CartQ and Bengali agent
- Demo videos, README cleanup
- DSPy (compressed to 3 days) + DSA practice

### Weeks 11–12: Interview Prep + Applications
- DSA practice (arrays, strings, hashmaps, trees)
- System design for AI systems
- Mock interviews
- Active job applications

## 🔄 Parallel Track (Every Week)

**Non-negotiable activities running alongside technical work:**

- **LinkedIn/X:** Post or comment 2–3x/week on AI engineering topics
- **DSA:** 30–45 min/day practice from Week 1 onward
- **Job market scan:** Weekly check of target companies (Brain Station 23, SELISE, BD AI job groups)

## 📊 Key Differentiators

This roadmap prioritizes:

- **Production-shaped artifacts** over tutorial completions
- **Evaluation-first mindset** (RAGAS, LangSmith) over just building
- **Real portfolio integration** (CartQ, Bengali agent) over generic assignments
- **Market-aligned skills** (Azure, LangGraph) over trending but less-demanded technologies
- **Interview readiness** (DSA, system design) often missing from AI curricula
